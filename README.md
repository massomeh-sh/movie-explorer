# 🎬 Movie Explorer

A modern movie discovery application built with **React**, **TypeScript**, and the **TMDB API**. Search for movies, explore detailed information, discover related titles, and save your favorites for later.

## ✨ Features

- 🔍 Search movies by title
- 🎥 View detailed movie information
- ⭐ Browse movie recommendations
- ❤️ Save favorite movies
- 📱 Responsive design
- ⚡ Fast and smooth user experience
- 🌐 Offline detection with user-friendly messages
- ⏳ Loading and error states

## 🛠️ Built With

- React
- TypeScript
- Vite
- Tailwind CSS
- Context API
- useReducer
- Custom Hooks
- TMDB API

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/movie-explorer.git
```

### Navigate to the project

```bash
cd movie-explorer
```

### Install dependencies

```bash
npm install
```

### Create a `.env` file

```env
VITE_TMDB_TOKEN=your_tmdb_access_token
```

You can get your API token from the TMDB website.

### Start the development server

```bash
npm run dev
```
